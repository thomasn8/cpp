1. enregistrer le script dans un directory mentionné dans le $PATH :

$ echo $PATH
/Users/thomas/.brew/bin:/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin

$ cp ClassName.sh /usr/local/bin

2. Installer une alias pour ./ClassName dans .zshrc

vim .zshrc

insert : 
alias class="ClassName.sh"

3. Utiliser cette alias (class) pour lancer la commande depuis n'importe où
